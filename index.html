GearUp

Interactive MR Automotive Maintenance Simulator | 2026

An immersive Mixed Reality tabletop simulator that puts a Perodua Myvi engine in your hands - without needing a real engine bay. GearUp uses Meta Quest 3S passthrough to layer virtual engine components onto your physical workspace.

  • Meta Quest 3S passthrough blending virtual components with the real world
  • Step-by-step guided assembly of Myvi engine components
  • Spatial interaction system for component grab, snap, and inspect
  • Educational overlay system with contextual tooltips
  • Built in Unity 6.3 using the Meta XR SDK
